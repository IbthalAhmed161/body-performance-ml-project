# Body Performance Classification System

## 📌 Overview
This project analyzes body performance data and builds machine learning models to classify fitness levels and predict physical performance.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Scikit-learn
- Machine Learning

## 📊 Dataset
- 13,393 rows
- 12 features (physical measurements)
- Target: Performance class (A, B, C, D)

## 🔍 Key Analysis
- Cleaned data and removed invalid records
- Handled outliers using IQR method
- Performed full EDA and visualization
- Created new features (BMI, strength index, etc.)

## 🤖 Models Used
- KNN
- Decision Tree
- Random Forest
- SVM
- Neural Network

## 📈 Results
- Best Classification Model: Random Forest
- Best Regression Model: Linear Regression
- R² Score: 0.79
